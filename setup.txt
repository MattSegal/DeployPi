===== Basic Linux Config ====

# Raspi config
sudo raspi-config
	* change password
	* expand filesystem
	* set to boot to command line

# Update then upgrade packages
	# download list of available packages, no installation
	sudo apt-get update

	# install new versions of packages
	sudo apt-get upgrade

===== Pull Setup =====

git clone https://github.com/MattSegal/DeployPi.git /home/pi/setup

===== Run Setup =====

cd /home/pi/setup/scripts

# Install apps
	* rdp
	* samba
	* kodi
	* dos2unix [2]

# Configure things
	* .bashrc

	* ssh
